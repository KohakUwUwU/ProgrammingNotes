## 模块和参数

粒子系统组件下有 23 个模块 **Module**

其中，

-   一个主模块 Main Module，**基础参数**
-   22个子模块，**扩展参数**

默认启用了3个子模块

-   Emission，发射的频率
-   Shape，粒子发生器的形状
-   Renderer，粒子的显示

还可以启用更多的子模块，如 Size over lifetime

基础参数：

在 Main Module 下，

-   Start Color，粒子的初始颜色
-   Start Speed，粒子的初始速度（m/s）
-   Start Size，粒子的初始大小（m）
